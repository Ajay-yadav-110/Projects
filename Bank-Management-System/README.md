# Bank Management System

This project is a console-based Bank Management System implemented in C++. It allows users to manage bank accounts efficiently through various operations.

## Features

* Create a new bank account
* Deposit amount into an account
* Withdraw amount from an account
* Display details of a specific account
* Display all accounts
* Update account information
* Delete an account
* Transfer amount between accounts

## Technologies Used

* C++ (Core Programming)
* File Handling (for storing account data)

## How to Run

1. Compile the program using a C++ compiler (e.g., g++)
2. Run the executable file
3. Use the menu to perform different banking operations

## Project Type

Console-based application
