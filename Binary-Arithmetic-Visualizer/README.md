# Binary Arithmetic Visualizer

This project is a web-based application that helps visualize binary arithmetic operations and algorithms in an interactive way.

## Live Demo

🔗 https://ajay-yadav-110.github.io/Projects/Binary-Arithmetic-Visualizer/

## Features

* Binary arithmetic visualization
* Booth’s multiplication algorithm
* Restoring division algorithm
* Non-restoring division algorithm
* Simple and interactive user interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Python

## File Structure

* index.html – Main webpage
* style.css – Styling of the webpage
* script.js – Frontend logic
* booths_algorithm.py – Booth’s algorithm implementation
* restoring_division_algorithm.py – Restoring division
* non_restoring_division_algorithm.py – Non-restoring division

## Project Type

Web-based educational tool
