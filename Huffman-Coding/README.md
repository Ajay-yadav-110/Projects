## Overview
This project implements the Huffman Coding algorithm in Python to perform lossless text compression. It assigns variable-length binary codes to characters based on their frequency, reducing the overall size of the data.

## Features
- Character frequency analysis
- Huffman tree construction using min-heap
- Generation of prefix-free binary codes
- Encoding and decoding of text
- Compression analysis (ratio, entropy, average code length)

## How to Run
1. Make sure Python is installed
2. Run the file using:
   python huffman.py
